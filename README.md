# VOCALISE
just a basic Text to Speech program made with gTTS

## INSTALLATION
linux Only
### 1
download the installer.run and vocaliser.zip to your computer
### 2
unpack the vocaliser.zip
### 3
run the installer.run
### 4
when done, just go to the upacked zip file in your terminal and run the command "electron ."
